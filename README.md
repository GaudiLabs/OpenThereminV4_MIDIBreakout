BreakOut board for the OpenTheremin V4.5 (featuring a 10 pin break out connector).
- MIDI DIN Connector
- Main Signals on soldering tabs (+5V / GND / AUDIO / CV1 / CV2 / GATE / TXD / RXD / RESET / GND
- Use a 2x5pin, 1.27mm pitch jumper cable to connect to main board
- Upload the OpenTheremin V4 MIDI software
